# Elijah She Yu Sheng | Data Engineering Portfolio

A personal portfolio website showcasing data engineering skills, academic projects, coursework portfolio, leadership experience, achievements, and contact links.

This is a static GitHub Pages website based on the Hyperspace template by HTML5 UP.

## Technologies

- HTML5
- CSS3
- JavaScript
- HTML5 UP Template
- GitHub Pages

## Project Structure

```text
.
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   ├── sass/
│   └── webfonts/
├── images/
├── README.md
├── README.txt
└── LICENSE.txt
```

- `index.html` contains the one-page portfolio sections and must remain in the repository root for GitHub Pages.
- `assets/` contains the website CSS, JavaScript, Sass source files, and Font Awesome webfonts.
- `images/` contains local project images used by the portfolio.
- `README.md` documents the completed portfolio website.
- `LICENSE.txt` and `README.txt` are retained from the original HTML5 UP template.

## Local Preview

Option 1: open `index.html` directly in a browser.

Option 2: run a local server from the project root:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages Deployment

1. Push the project files to the GitHub repository.
2. Go to `Settings > Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Set `Branch` to `main`.
5. Set `Folder` to `/root`.
6. Save.
7. Wait for GitHub Pages to publish the site.

Important: `index.html` must stay in the root folder.
